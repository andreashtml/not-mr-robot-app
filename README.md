# Not Mr.Robot App

![License](https://img.shields.io/badge/license-MIT-brightgreen)
![CI](https://github.com/andreashtml/not-mr-robot-app/actions/workflows/ci.yml/badge.svg)

Simple Node.js blog that serves Markdown posts from the `posts` folder. You can list posts, view them, and add new ones via a form.

Repository: https://github.com/andreashtml/not-mr-robot-app

Notes

- Posts are stored as `.md` files in the `posts` folder.
- New posts are created via the web form at `/new`.
